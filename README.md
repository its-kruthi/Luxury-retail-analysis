# Luxury Retail Sales Dashboard
End-to-end sales analytics project for a luxury boutique chain across 5 Indian metro cities — built using MySQL, Excel, and Power BI.

## Tech Stack
`MySQL` `Excel` `Power BI` `GitHub`

## Dataset
- 240 transactions | Jan–Aug 2024
- 5 cities: Mumbai, Delhi, Bengaluru, Chennai, Hyderabad
- 6 product lines | 3 customer segments (Premium / Regular / Budget)

## What I Built
- **MySQL:** Database setup, data import, and 5 business-driven SQL queries
  (monthly trend, product performance, city revenue, quarterly KPIs, 
  customer segmentation)
- **Excel:** MIS report with KPI tiles, Pivot Tables, and charts
- **Power BI:** Interactive dashboard with DAX measures, Date Table, 
  and slicers for City, Product Line, and Customer Segment

## Key Insights
- Electronics drives the highest revenue across all cities
- Delhi and Mumbai together contribute ~49% of total revenue
- Premium segment generates the highest revenue per transaction
